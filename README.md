# Student Database Management System

This repository contains a menu-driven Student Database Management System developed using the C programming language.

---

## Description

The Student Database Management System is a console-based application that allows users to store and manage student records using file handling.  
It supports basic database operations through a simple text-based menu.

---

## Functionality

The program allows the user to:

- Add new student records
- Display all student records
- Search a student by roll number
- Update existing student information
- Delete student records
- Store data permanently using files
